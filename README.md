# Unit testing

- Simple unit test template exercise using Mocha and Chai.

## What the repository contains?
### In scripts folder
- mylib.js contains functions of basic arithmetic operations
- main.js contains very simple main program, that prints outputs only in console. Existing only to show that tests run without it.
### In test folder
- mylib.test.js file, contains the test script

## Usage

### Initalization
- npm init -y

### Depencies installation
- npm install --save-dev mocha chai
